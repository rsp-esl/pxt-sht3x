# SHT3X

A MakeCode/PXT extension for the Sensirion's SHT3x digital temperature and humidity sensor.

![](icon.png)

## Wiring

Pins (GY-SHT3x module <--> Microbit Edge Connector)
- Vcc: 3V pin (3.3V)
- SCL: Pin 19
- SDA: Pin 20
- Gnd: Gnd pin (0V)
